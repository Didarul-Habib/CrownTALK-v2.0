===============================================================================
0) Executive snapshot
===============================================================================
Wallchain presents itself as an “ecosystem” with (at least) two major active product
tracks:
  A) AttentionFi: an AI-driven growth + reputation system for Crypto Twitter (X)
     that measures “mindshare” and rewards high-quality, consistent social
     contributions (X Score + Quacks). The system is designed to resist bots,
     farming, and “KOL cabal” self-engagement loops. [Sources: docs + website]
  B) Wallet / infra: “Native Yield Abstraction (NYA)” for wallets/apps using
     Account Abstraction (ERC-4337 via ZeroDev) + session keys to automatically
     deposit idle assets into curated yield strategies and withdraw on demand.
     [Sources: docs]

Wallchain also highlights prior work on MEV value capture & redistribution and
open-source “meta-intents”, positioning that track as part of their credibility
and/or a parallel product line. [Sources: team page + GitHub + Binance program post]

Core promise (in plain English):
  - For creators + communities: earn recognition (and potentially rewards) by
    being genuinely influential and consistently helpful—measured via algorithms,
    not vanity metrics.
  - For brands: run mindshare campaigns and discover authentic influencers.
  - For wallets/apps: unlock a “savings-account-like” yield layer on idle assets
    with minimal UX friction.

1) What problem Wallchain targets
===============================================================================
1.1 AttentionFi (social layer)
- Traditional “influence” and “analytics” tools often overfit to follower counts,
  bot-driven engagement, and superficial virality.
- Marketing spend in crypto is vulnerable to low-quality accounts, farming, and
  misaligned incentives.
Wallchain’s docs emphasize: quality-weighted attention (“mindshare”), loyalty,
anti-gaming, and optionally onchain footprint.

1.2 Native Yield Abstraction (wallet layer)
- Idle assets in wallets generally earn nothing; wallet providers have limited
  monetization options beyond swaps and on/off-ramp fees.
Wallchain frames NYA as a way to turn idle balances into passive yield without
“locking” usability—deposit/withdraw happens automatically via AA.

Key references:

1.3 MEV (execution layer — historical / parallel)
- MEV extraction (sandwiching/frontrunning/backrunning) can worsen user execution
  and siphon value.
Wallchain has historically focused on capturing MEV and redistributing a portion
as user cashback/rebate, integrated by DEXs such as PancakeSwap/QuickSwap (as
claimed by Wallchain team docs).

2B) Native Yield Abstraction (NYA) + Account Abstraction infrastructure
-------------------------------------------------------------------------------
High-level proposition
- Users earn yield “just by holding assets” in an integrated wallet; docs suggest
  5–10% annually (presented as “potentially”). citeturn11search14turn11search12
- Wallet providers monetize idle assets as a new revenue stream. citeturn11search14turn11search12

Core architecture (as described)
- Account Abstraction via ZeroDev (Kernel implementation mentioned). citeturn11search14turn9search12
- Each user’s EOA maps 1-to-1 to an AA account, created “under the hood”. citeturn10view6
- Session keys enable permissioned automation and can be revoked. citeturn11search14turn10view6turn9search12
- SDK converts normal transactions into UserOperations (UserOps), supports batch
  and simulation. citeturn10view6
Source: https://docs.wallchain.xyz/infra/quickstart citeturn10view6

Security posture / guarantees (as described)
- Self-custodial; no access to private keys/seed phrases; user owns smart account. citeturn9search12
- Auto-withdrawal / emergency withdrawal if onboarded protocol is compromised. citeturn9search8turn10view7
- Proposal mentions “replacement guarantee” and “select insurance coverage”

Source: https://docs.wallchain.xyz/wallchain-service-overview/proposal citeturn10view7
