ONE_LINE_PITCH:
Wallchain is an AI-powered AttentionFi ecosystem that measures and rewards authentic influence in crypto through content creation and engagement on X, using X Score (0–1000) + Quacks points + Mindshare leaderboards, with wallet-based multipliers and historical MEV/cashback infrastructure on EVM.

CORE_NARRATIVE:
Wallchain addresses fake engagement and low-quality influence in Web3 by building a quality-first reputation + rewards layer.

Key narrative pillars (combined):
- Fake engagement problem: bots, engagement pods, AI slop → follower count is noisy
- Quality-weighted influence: measure who engages with you (smart followers + smart interactions), not just raw numbers
- Mindshare as measurable signal: quantify who drives meaningful conversation around projects/people
- Two core components:
  1) Wallchain X Score: 0–1000 crypto influence score using a Crypto Twitter Connectivity Graph (CTCG)
  2) Wallchain Quacks: daily-evaluated points rewarding valuable/insightful/engaging content + meaningful engagement
- “Attention layer of Web3”: gives projects a way to distribute rewards fairly to creators/KOLs using measured mindshare (instead of pure hype)
- Compounding model: consistent high-quality participation builds lasting reputation + campaign eligibility
- Evolution arc:
  - Earlier phase: MEV/intent/cashback infrastructure
  - Current phase: creator growth + reputation + mindshare campaigns (AttentionFi)

Backed/accelerators positioning (combined):
- Backers listed in Wallchain docs include AllianceDAO, NGC, OrangeDAO, Woodstock, MH Ventures, Cypher Capital, Skyland, LBank, and others
- Founders mention going through programs including Y Combinator, Alliance DAO, Orange DAO, and BNB Chain accelerator

PRODUCT_DETAILS:
Wallchain offers a dual-product suite (merged):

A) AttentionFi Ecosystem (Quacks.app / app.wallchain.xyz)
- What it is:
  - Platform where projects run mindshare leaderboards/campaigns and users (“Quackers”) earn Quacks for meaningful content + engagement.
- Core primitives:
  1) X Score (0–1000)
     - Derived from Crypto Twitter Connectivity Graph (CTCG): retweets, follows, comments
     - Uses “seed accounts” proximity + exponential decay to reduce manipulation
     - Documented behaviors:
       - Scores never decrease based on user actions
       - Influence “passes” via interactions (others don’t lose influence)
       - X Score ~100 is described as roughly top 1% of Crypto Twitter
  2) Quacks (Points)
     - Daily-calculated points rewarding valuable/insightful/engaging content and meaningful engagement
     - Anti-farming: bot/farming behavior can trigger permanent bans
  3) Mindshare Leaderboards
     - Rank users based on “mindshare” (quality-weighted attention)
     - “All types of public engagement count”
     - Time filters commonly include: 7d / 30d / epochs (and often longer windows like 3 months / all time on Quacks page)
- UX / mechanics (combined):
  - Login with X → profile shows X Score + Quacks balance + invite/referral tools
  - Personal mindshare leaderboard:
    - Top 5 supporters visible immediately
    - Full list unlock via in-app flow (share action mentioned in Wallchain blog)
    - Includes auto-generated “vibe descriptions” for supporters (blog)
  - Quacks page shows:
    - Top quackers + your Quacks + X Score + earned + referral Quacks
    - Filters: 7 days / 30 days / 3 months / all time
  - Partner leaderboards show:
    - Project context + team + socials + featured tweets
    - Filters: 7 days / 30 days / epochs
  - Referral:
    - Earn 10% of Quacks earned by referrals (points share)
  - Wallet / multipliers:
    - “Unified wallet connection” (connect once, use across projects)
    - Wallets checked for on-chain activity across leaderboards
    - Visual on-chain multipliers live for some leaderboards
    - Some campaigns explicitly tell users to add Solana + EVM wallets

RISKS_AND_REALISM:
(merged full list)

Market/platform dependency:
- Reliance on X (Twitter) data access + platform policies; API/rate-limit changes can disrupt scoring and campaign mechanics

Competition:
- Competes with SocialFi/AttentionFi and “creator scoring / campaign” platforms; must maintain differentiation via quality and anti-farm enforcement

Centralization / model risk:
- Wallchain controls algorithm parameters, seed accounts logic, multipliers, campaign rules, bans/enforcement; ranks can shift with updates

Incentive design risk:
- Reward systems attract farming; “AI/LLM scoring” can create false positives/negatives; quality scoring is partly subjective

Token launch uncertainty:
- No confirmed native token in official docs; any token speculation can create volatility and misinformation risk
- High impersonation risk: third-party “QUACKS presale” pages exist; verify only via wallchain.xyz/docs links

Regulatory/compliance:
- Campaign reward distribution across jurisdictions can invite scrutiny depending on partner token rules

Smart contract / infra risk (legacy products):
- Audits improve confidence but don’t eliminate risk; wrappers and integrations can include privileged functions and integration risk

Business/PMF realism:
- Strong traction signals via campaigns and reported payouts; long-term monetization strategy beyond partner campaigns is not fully public
